# Project Context

## Project Name

TaskFlow - Task Management Portal

## Purpose

This project is being built as a Full Stack Developer internship assessment.

The objective is to demonstrate:

* React fundamentals
* Backend API development
* Database design
* API integration
* Clean architecture
* Clean code
* Documentation

The project should feel like a small production SaaS application.

---

## Users

A registered user can:

* Register
* Login
* Create tasks
* View tasks
* Update tasks
* Complete tasks
* Delete tasks
